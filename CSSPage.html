Changing a Background Color

After creating a simple webpage, colors can be added to follow a theme. In order to add colors, we can use CSS to achieve that goal.

First we create our styles.css file and link it to our HTML like we learned in the previous lesson. Once that has been created and linked, we can begin making changes.

If we want to change our background to gray, we would add that to the body of the page and apply the changes. Colors can be chosen using RGB (red, green, blue) properties or Hex (#). In this example we wanted to targe the body of the page and we used the hex option and the color is #bcbcbc.

Here's what that looks like in CSS:

                body {
                    background-color: #bcbcbc;
                }

Changing the Font Size

Font size can be changed to differentiate between sections. You always want the title of the page to be larger than the remaining text on your webpage but you also want each section to stand out.

Fonts can be changed within HTML using tags as we learned in the previous lesson but CSS can be used if you're looking for a more specific size. This can be done using pixels (px) or percentage (%) sizes.

In this example, we set a class to the first line and used a period (.) to target each class in CSS. For the class "line1" we used .line1 in CSS to target the font size and make it 25px. If you use an id instead of a class, this would be targeted with a hashtag (#) which we can see in the example as #line2 with a font size of 40px.

Here's what that looks like in CSS:

                body {
                    background-color: #bcbcbc;
                }
                
                .line1 {
                    font-size: 25px;
                }
                
                #line2 {
                    font-size: 40px;
                }

Creating a Box Shadow

Placing boxes around content can bring attention to a section. In addition to adding a box around the text and editing the text, the box can also be adjusted by adding a shadow.

A shadow can be added to a specific class or id using the "border" and "box-shadow" properties in CSS. By using "border" you are creating a solid line around the text and the width can be altered with pixels. Once the border has been added, the box-shadow property can be used to create a shadow effect. There are three values in this property: horizontal offset, vertical offset, and blur radius.

In the example provided, we can see the box-shadow and border properties were added to the class .example1 but only the border property was added to the id #example2 and it does not have a shadow.

Here's what that looks like in CSS:

                    body {
                        background-color: rgb(255,255,255);
                    }
                    
                    .example1 {
                        display: inline-block;
                        border: 1px solid #ccc;
                        box-shadow: 0px 4px 8px #444444;
                        padding: 20px;
                    }
                    
                    #example2 {
                        display: inline-block;
                        border: 3px solid #ccc;
                        padding: 20px;
                    }

Changing the Text Color

Text color is very important when you are designing a webpage. It is important to make your site accessible to all who visit. If the colors chosen don't have enough contrast with the background, it can make it difficult to read what you share.

Like choosing a background color, we can use the hex(#) or RGB option to select a color for our text. Here we can see the use of both.

In this example, we can see four different colors against a white background. The title is a shade of brown #736653 and it stands out. Directly below it, though, a very light shade of blue #dbece6 was used and the text is not very visible. The third line uses a slightly darker shade #97a697 but it could still be a bit darker. The last line is in rgb (black) and it is the easiest line to read from all of those examples.

It is good practice to test out colors prior to deploying your site and to make sure the content will be visible.

Here's what that looks like in CSS:

                body {
                    background-color: rgb(255,255,255);
                    color: #736652;
                }
                
                .example1 {
                    color: #dbece6;
                    font-size: 25px;
                }
                
                #example2 {
                    color: #97a697;
                    font-size: 25px;
                }
                
                .example3 {
                    color: rgb(0,0,0);
                    font-size: 25px;
                }
